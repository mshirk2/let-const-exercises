Quiz
What is the difference between var and let?
    var is applied to the window and can be reclared, let has a block scope and cannot be redeclared
What is the difference between var and const?
    var has a function scope and can be reassigned, const has a block scope and cannot be reassign
What is the difference between let and const?
    let can be reassigned, const cannot
What is hoisting?
    hoisting is when the browser reads var first when initializing the code
